# QingClaw Skills × 轻流 50 Skills

> 以 OpenClaw 的 AI Agent 能力 + [轻流 10.0](https://qingclaw.oalite.com/)「搭·问·析·生·联」五大 AI 能力矩阵 + QingClaw MCP，覆盖企业 **10 大业务领域 × 5 = 50 个 Skills**，形成 "AI 生成 + 无代码搭建 + 数据驱动 + 流程自动化" 的完整智能闭环。

---

## 轻流 MCP 三件套

| 包名 | 用途 | 安装命令 |
|------|------|---------|
| `@josephyan/qingflow-cli` | CLI 命令行工具（搭建、发布、调试） | `npm install @josephyan/qingflow-cli@beta` |
| `@josephyan/qingflow-app-user-mcp` | 运行态能力（数据读写 / 任务 / 分析 / 导入导出） | `npm install @josephyan/qingflow-app-user-mcp@beta` |
| `@josephyan/qingflow-app-builder-mcp` | 搭建态能力（建模 / 流程 / 视图 / 图表 / 门户） | `npm install @josephyan/qingflow-app-builder-mcp@beta` |

## 目录结构

```
skills/
├── 01-HR/                  # 人力资源
├── 02-Finance/             # 财务管理
├── 03-Operations/          # 运营管理
├── 04-Sales/               # 销售管理
├── 05-Marketing/           # 市场营销
├── 06-Engineering/         # 技术研发
├── 07-Product/             # 产品管理
├── 08-Legal/               # 法务合规
├── 09-ProjectMgmt/         # 项目管理
└── 10-AdminIT/             # 行政与IT
```

每个 Skill 以独立 Markdown 文件呈现，包含：功能介绍、Prompt 模板、轻流能力标签、安装命令、协作链路等。
