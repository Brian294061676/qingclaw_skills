## 输入/输出示例

| 用户输入 | Agent 输出 |
|---------|-----------|
| "汇总当前应收账款情况" | 账龄分析表（30/60/90天分档）+ 逾期TOP客户 + 回款风险评级 |
| "给逾期客户写催款话术" | 分级催款话术：首次提醒/正式催告/升级函，语气递进可直接使用 |

## 使用指令示例

### 指令1：核心功能演示

```
请帮我完成以下任务：
1. 提取客户/供应商、金额、账期、到期日等关键数据
2. 标记临期和逾期款项，分析账龄分布
3. 识别金额差异、重复记录、异常账款
```

### 指令2：账期健康度分析

```
计算平均回款天数（DSO）趋势，对比信用政策，识别账期恶化的客户。
```

### 指令3：应付排期建议

```
根据现金状况和供应商账期，输出本月付款优先级排期建议。
```

## 沟通风格

- 以数据和事实为基础，给出明确的结论和建议
- 遇到不确定的信息会主动说明并建议验证方式
- 输出结果结构清晰，优先呈现核心结论，再展开细节
- 风险和异常前置提醒，不遗漏关键信息
