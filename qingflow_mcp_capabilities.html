轻流当前已支持的 MCP 能力清单

本页基于当前会话中可直接调用的 qingflow 与 qingflow_builder MCP 工具整理， 重点覆盖认证、工作区、应用、记录、任务、导入导出、流程、视图、图表、门户等能力。

覆盖运行态 + Builder 能力 适合做能力盘点与内部说明 整理日期：2026-04-13

1. 认证与工作区

支持登录态管理、Token 接入、身份确认与工作区切换。

  • 账号登录 / 退出：auth_login、auth_logout
  • Token 接入：auth_use_token
  • 查看当前身份：auth_whoami
  • 列出工作区 / 切换工作区：workspace_list、workspace_select
  • Builder 侧同样支持独立认证与工作区切换

2. 应用与门户读取

支持读取业务应用、视图、图表、门户等配置与摘要信息。

  • 应用查询：app_list、app_search、app_get
  • 视图读取：view_get、Builder app_get_views
  • 图表读取：chart_get、Builder app_get_charts
  • 门户读取：portal_list、portal_get
  • Builder 支持按名称或 key 解析应用：app_resolve

3. 组织架构与人员目录

支持部门、内部成员、外部联系人等目录信息读取与搜索。

  • 部门列表 / 子部门：directory_list_internal_departments、directory_list_sub_departments
  • 全量部门树：directory_list_all_departments
  • 内部成员列表：directory_list_internal_users、directory_list_all_internal_users
  • 外部联系人列表：directory_list_external_members
  • 统一目录搜索：directory_search
  • Builder 侧成员搜索：member_search

4. 记录读取与浏览

支持 schema-first 方式读取单条记录、列表记录和浏览 schema。

  • 读取单条记录：record_get
  • 记录列表浏览：record_list
  • 浏览 schema：record_browse_schema_get
  • 插入 schema：record_insert_schema_get
  • 更新 schema：record_update_schema_get
  • 适合做字段探查、记录抽样、导出前确认

5. 记录新增、更新、删除

支持在运行态对业务数据做标准 CRUD。

  • 新增记录：record_insert
  • 更新记录：record_update
  • 删除记录：record_delete
  • 支持写前预校验与写后验证
  • 支持 relation / member / department 等字段的自然输入解析

6. 记录分析能力

支持基于字段 schema 的分析型查询，适合做统计、分组、聚合。

  • 分析入口：record_analyze
  • 支持整表汇总、分组统计、排序、筛选
  • 适合做数量、占比、排行、趋势类分析
  • 属于“最终结论类”能力，不只是浏览明细

7. 导入与导出辅助

支持模板获取、校验、修复、本地导入启动与状态跟踪。

  • 获取导入模板：record_import_template_get
  • 导入 schema：record_import_schema_get
  • 导入校验：record_import_verify
  • 本地修复导入文件：record_import_repair_local
  • 启动导入 / 查询状态：record_import_start、record_import_status_get

8. 文件与附件

支持获取上传信息、本地文件上传与附件字段配合使用。

  • 获取上传信息：file_get_upload_info
  • 运行态上传本地文件：file_upload_local
  • Builder 侧上传本地文件：builder.file_upload_local
  • 适合图片、附件、导入模板等场景

9. 任务与流程运行态

支持待办查询、任务上下文、流程日志、动作执行等运行态能力。

  • 任务列表 / 待办箱：task_list
  • 任务详情：task_get
  • 流程日志：task_workflow_log_get
  • 关联报表明细：task_associated_report_detail_get
  • 执行任务动作：task_action_execute

10. 成员候选与字段候选集

支持在 schema 基础上查出成员字段、部门字段的候选范围。

  • 成员候选：record_member_candidates
  • 部门候选：record_department_candidates
  • 适合做可选范围确认、自动填充前探查

11. 代码块与远程联动

支持运行表单代码块、读取代码块 schema，并支持 Builder 配置代码块/QLinker。

  • 读取代码块 schema：record_code_block_schema_get
  • 运行代码块：record_code_block_run
  • Builder 可配置 code_block 与 q_linker 字段
  • 适合做计算字段、外部查询回填、复杂表单联动

12. Builder 建模能力

支持在 Builder 侧直接创建和维护应用结构。

  • 应用创建 / 更新 schema：app_schema_apply
  • 表单布局调整：app_layout_apply
  • 流程配置：app_flow_apply
  • 视图配置：app_views_apply
  • 图表配置：app_charts_apply
  • 发布校验：app_publish_verify

13. Builder 门户与解决方案

支持门户、解决方案包、应用归档到包等能力。

  • 创建 / 更新门户：portal_apply
  • 读取门户：builder.portal_get、builder.portal_list
  • 包创建 / 查询：package_create、package_list、package_resolve
  • 应用挂到包：package_attach_app

14. Builder 角色与发布辅助

支持创建角色、搜索角色、编辑锁处理等 Builder 侧辅助操作。

  • 角色创建 / 搜索：role_create、role_search
  • 自定义按钮：app_custom_button_create、app_custom_button_update 等
  • 发布锁释放：app_release_edit_lock_if_mine
  • 代码块修复：app_repair_code_blocks

15. 当前可总结为 5 大类能力

  • 运行态数据能力：读、写、删、分析、导入、文件上传、任务执行。
  • 组织与权限支撑能力：成员目录、部门目录、成员候选、部门候选、角色管理。
  • Builder 建模能力：应用 schema、布局、流程、视图、图表、门户、解决方案包。
  • 自动化扩展能力：代码块、QLinker、流程动作、图表/门户展示。
  • 实施辅助能力：反馈提交、导入修复、编辑锁释放、发布校验、包管理。
说明：本清单按“当前会话里可直接调用的 MCP 工具”整理，因此是“当前已接入并可用的能力范围”，不是轻流所有潜在平台能力的完整白皮书。
