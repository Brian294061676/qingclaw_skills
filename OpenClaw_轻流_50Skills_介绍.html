安装与配置

两种方式快速上手 OpenClaw × 轻流 Skills

📄 方式一：直接安装 MD 文件

1

获取 Skill 文件

每个 Skill 均为独立的 .md 文件，包含完整的身份定义、能力描述和使用指令

2

发送给 OpenClaw

将 .md 文件直接发送给 OpenClaw（小龙虾）即可安装使用

3

开始使用

通过自然语言指令调用 Skill，AI Agent 自动执行对应能力

🔗 方式二：连接轻流 MCP

1

安装轻流 MCP 包

npm install @josephyan/qingflow-cli@beta npm install @josephyan/qingflow-app-user-mcp@beta npm install @josephyan/qingflow-app-builder-mcp@beta
2

认证登录

方式A：账号密码登录 → auth_login
方式B：Token 接入 → auth_use_token

3

选择工作区

workspace_list → workspace_select
4

验证连接

auth_whoami → 确认身份和工作区
